These fabrications files are meant for JLCPCB so I don't know if it will work with other fabricators.
It seems to work with NextPCB.

Also I'm too lazy to add anymore LCSC part numbers to the JLCPCB BOM so you will have to do most of them yourself.


The ESC.zip file is the gerbers.
The positions.csv is the pick and place file.
The bom.csv file is the bill of materials file.
and idk about everything else.

2 oz copper weight is recommended but it is fine to choose 1 oz if you're on a budget.
